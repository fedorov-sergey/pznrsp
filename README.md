## 🚀 Быстрый старт

### Где смотреть сайт:
- **GitHub Pages:** https://ТВОЙ_НИК.github.io/ТВОЙ_РЕПОЗИТОРИЙ
- После каждого коммита сайт обновляется автоматически (может занять 1-2 минуты)

### Как редактировать:

1. **Изменить ссылки на скачивание:**
   - Открой `index.html`
   - Найди комментарии `<!-- ЗАМЕНИ # НА ССЫЛКУ GOOGLE ДИСК... -->`
   - Замени `#` на реальные ссылки

2. **Изменить ссылку на Boosty:**
   - В `index.html` найди `<!-- ЗАМЕНИ # НА ССЫЛКУ BOOSTY -->`
   - Замени `#` на свою ссылку

3. **Добавить ссылки на соцсети:**
   - В `index.html` найди блок `<!-- БЛОК СОЦСЕТЕЙ -->`
   - Замени `#` на ссылки своих каналов

4. **Добавить видео-инструкции:**
   - Открой `videos.html` (создается отдельно)
   - Добавь новую строку в таблицу
