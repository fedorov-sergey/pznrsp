Как оформить подписку
1
Проверьте ID

Убедитесь, что ID на экране компьютера совпадает с ID выше. Он будет передан разработчику для создания лицензионного ключа.

2
Скопируйте и отправьте

Нажмите кнопку «Копировать ID», затем отправьте его разработчику через любую удобную социальную сеть (ссылки ниже).

3
Получите ключ

Разработчик подготовит персональный ключ активации и пришлёт его вам. Установите файл ключа в приложение.
